Descreve 3 benefícios de utilizar uma arquitetura orientada a eventos.
Os eventos são assíncronos, isso permite que os recursos se movam entre as tarefas.
Escalonamento fácil, serviços são separados em uma arquitetura de eventos quando há necessidade de algum processamento mais volumoso é possível o escalonamento de apenas um serviço específico. 
Maior apoio a recuperação de falhas, como exemplo nos broker de mensageria quando a uma falha em uma tarefa é possível enviar para outra fila para processamento posterior.
